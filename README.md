# desaign-team-garage

> "design" に "ai" を仕込んだ名前です。デザイナーがAIとフロントエンドを学びながら、手を動かしてひとつのWebアプリをつくるプロジェクトです。

## 🚧 ステータス

構想中・学習中です。内容は今後どんどん更新していきます。

## このプロジェクトについて

普段デザイン業務をしているメンバーが、AIツール（Claude Codeなど）とフロントエンドの基礎（HTML / CSS / JS）を、実際に手を動かしながら学ぶための個人プロジェクトです。

学んだことをメモとして溜め込むだけでなく、最終的には1つの動くWebアプリという形に残すことをゴールにしています。

「デザインはできるけど、コードはまだ自信がない」「AIツールをどう開発に活かせるのか知りたい」という気持ちからスタートしました。

## ゴール

- HTML / CSS / JS の基礎を理解し、自分の手でWebページを組み立てられるようになる
- Claude CodeなどのAIツールを使った開発の進め方を体験する
- 学びの過程を、実際に動くアプリという成果物の形に残す

## ローカルで見る方法

React版のサイト（`web-react/`）は以下の手順でローカルに立ち上げて確認できます。

```bash
cd web-react
npm install   # 初回のみ
npm run dev
```

ターミナルに表示されるURL（例: `http://localhost:5173`）にブラウザでアクセスすると確認できます。停止する場合はターミナルで `Ctrl + C` を押してください。
